# Video Task Project

This project is a single-file internal video task management app.

Source of truth:

- v4.html contains the current full feature set.

Rules:

- Keep all current features.
- Do not remove Overview, Calendar, Video table, filters, sidebar, or task modal.
- Do not change business logic unless requested.
- Improve UI/UX only.
- Keep Vietnamese labels.
- Keep vanilla HTML/CSS/JavaScript and Tailwind CDN.
- Do not migrate to React/Vue.
- Before editing, analyze first.
- After editing, check for JavaScript errors and responsive issues.

Design direction:

- Premium internal SaaS dashboard.
- Clean, serious, modern, production-ready.
- No childish colors.
- No random gradients.
- No oversized demo cards.
- Better spacing, typography, hierarchy, table density, modal layout, and mobile responsiveness.
